常见于：常见于两个界面，一个用于输入数据（注入的有问题的SQL代码），另一个界面对数据查询（查询注入结果）。

addslashes的定义和用法

addslashes() 函数返回在预定义字符之前添加反斜杠的字符串。
预定义字符是：

  • 单引号（’）
  • 双引号（”）
  • 反斜杠（\）
  • NULL

简单说就是转义这四个字符的函数。
